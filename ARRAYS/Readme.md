# Arrays in C++

Arrays are one of the most fundamental data structures. They store elements of the same data type in contiguous memory locations and allow efficient access using indices.

---

## Time Complexities

| Operation | Time Complexity |
|------------|----------------|
| Access | O(1) |
| Search | O(n) |
| Insertion (End) | O(1) |
| Insertion (Middle) | O(n) |
| Deletion | O(n) |
