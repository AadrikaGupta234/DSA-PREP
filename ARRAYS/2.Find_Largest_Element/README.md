# Find Largest Element in an Array (C++)

## Description

This program finds the largest element in an integer array using a linear traversal approach.

The algorithm iterates through the array, compares each element with the current largest value, and updates the largest value whenever a greater element is found.

---

## Algorithm

1. Assume the first element is the largest.
2. Traverse the array from the second element to the last.
3. Compare each element with the current largest element.
4. Update the largest element if a larger value is found.
5. Return or print the largest element.

---

## Time Complexity

* **O(n)**

The array is traversed only once.

## Space Complexity

* **O(1)**

No extra space is used apart from a few variables.
