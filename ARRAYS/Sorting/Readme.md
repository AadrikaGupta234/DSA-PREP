# Sorting Algorithms

This project implements common sorting algorithms:

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

---

## Bubble Sort
Swaps adjacent elements repeatedly.

## Selection Sort

Selects minimum element each pass.

## Insertion Sort

Builds sorted array one item at a time.

## Merge Sort

Divide and merge sorted halves.

## Quick Sort

Pivot-based partitioning.

Complexity(Summary)
Algo	  |   Avg Time
__________|____________
Bubble	  |      O(n²)
Selection |      O(n²)
Insertion |	     O(n²)
Merge	  | O(n log n)
Quick	  | O(n log n)
